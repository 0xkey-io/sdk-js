## Introduction

The [`@0xkey-io/indexed-db-stamper`](https://www.npmjs.com/package/@0xkey-io/indexed-db-stamper) package, while sharing a similar purpose with the `@0xkey-io/api-key-stamper`, caters specifically to the unique context of IndexedDB. This package is designed for stamping requests through leveraging IndexedDB, using credentials for ZeroXKey's API
